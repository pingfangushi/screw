该项目根据[screw][screw]由来,核心代码没改动,旨在通过 idea 插件快速创建数据库文档,只需简单配置数据库连接信息即可使用

# 使用方法
1. 安装screw插件
2. Tools(或者编辑界面右键) --> Screw --> 选择对应数据库 --> 配置数据库信息 --> 确定 --> 选择文件路径 
3. 生成失败可以查看Event log 查看原因
